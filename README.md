## نبذة عن الشركة

**قرى AI (Qoura AI)** شركة إماراتية متخصصة في تمكين المجتمع والمؤسسات بالذكاء الاصطناعي عبر برامج تدريب تطبيقية، واستشارات مؤسسية، وحلول تقنية متقدمة مثل AI Agents والأتمتة وتحليل البيانات. تعمل الشركة بأسلوب مبسط وعملي باللغة العربية وبفهم عميق للثقافة المحلية الإماراتية.

## محتويات المستودع

| المجلد | الوصف |
|:---|:---|
| `01-Company-Profile/` | الملف التعريفي للشركة (Markdown + PDF) |
| `02-Brand-Guidelines/` | دليل الهوية البصرية الشامل (Brand Guidelines) |
| `03-Brand-Assets/` | ملفات البراندنج البصرية (بطاقات، ترويسة، سوشيال ميديا، إلخ) |
| `04-Landing-Page/` | كود موقع الشركة (Landing Page) |

## الخدمات الرئيسية

**برامج التدريب وبناء القدرات** — برامج تدريبية تطبيقية في الذكاء الاصطناعي للجهات الحكومية والخاصة والأطفال والناشئة.

**الاستشارات المؤسسية** — تصميم خارطة طريق للتبني، وبناء سياسات الاستخدام المسؤول، وتطوير جاهزية الفرق.

**الحلول التقنية** — تطوير AI Agents داخلية، وأتمتة الإجراءات، وتحليل البيانات الذكي.
